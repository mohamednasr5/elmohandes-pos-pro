# المهندس للمبيعات (Al Mohandes POS Pro)

نظام نقاط بيع متطور ومربح هام الما يعمل مع Firebase Realtime Database

## عن المشروع

هذا مشروع نظام تجاريط شامل متعدد العملاء (مع نظام اشتراكات شهرية) يوفر:

### الميزات الرئيسية:

1. **نظام الاشتراكات**
   - اشتراكات شهرية
   - لكل متجر بيانات منفصلة
   - زرخ الدي للمشتركين

2. **باركود وQR**
   - دعم باركود للمنتجات
   - متوافق مع أجهزة الباركود

3. **الطباعة**
   - دعم الطابعات العادية
   - دعم الطابعات الحرارية

4. **إدارة المخزون**
   - تحديث موادي بره راديبات
   - تنبيهات المخزون

5. **إدارة العملاء (CRM)**
   - قاعدة بيانات العملاء
   - نقاط الولاء

6. **التقارير**
   - تقارير يومية وشهرية
   - رلابز الربح والخسارة
   - أعلى المنتجات

## التركيبة

### مطالبيب يسابقة
- ماصر مع Firebase Realtime Database
- Node.js و npm

### الخطوات

1. **استنسخ المرصد**
   ```bash
   git clone https://github.com/mohamednasr5/elmohandes-pos-pro.git
   cd elmohandes-pos-pro
   ```

2. **بو Firebase**
   - اعتر متط Firebase
   - حدث `js/firebase-config.js` بجب Firebase

3. **إارقه الخادمة
   ```bash
   npm install
   npm start
   ```
